TC-SYS-007 — Anonymize and restore text via CLI

RequirementSYS-007
Test Date2026-02-23T12:13Z
TesterSybil Oracle (automated)

Procedure

  1. Run CLI anonymize inside Docker container.
  2. Run CLI restore with anonymized output.

Results

Step 1 — Anonymize:
$ docker exec pii-anonymizer-sidecar-pii-anonymizer-1 \
    python -m cli.main anonymize --session cli-test1 \
    --text 'Alice Smith works at Acme Corp'

Output: «PERS_1» works at «ORG_1»

Step 2 — Restore:
$ docker exec pii-anonymizer-sidecar-pii-anonymizer-1 \
    python -m cli.main restore --session cli-test1 \
    --text '«PERS_1» works at «ORG_1»'

Output: Alice Smith works at Acme Corp

Analysis

  • ✅ Anonymize stdout contains placeholders, no original entity values
  • ✅ Restore stdout identical to original input
